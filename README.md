# gr-car-schedule

車屋プロジェクト専用 予定管理 PWA。

- **フロントエンド**: 静的 HTML + JS（GitHub Pages 配信）
- **バックエンド**: Google Apps Script（独立 GAS Web App）
- **データ保管**: Google スプレッドシート（バインドスクリプト）
